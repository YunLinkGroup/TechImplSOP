- 命名空间：**mid**（与 RabbitMQ 同）
- 本地路径 `/storage/nacos`
- NodePort 暴露 **8848 → 30848**（HTTP 注册中心）、**9848 → 30849**（gRPC 客户端通信）
- 启用 **embedded 存储**（单节点免外置 MySQL），如需集群可再扩
- 默认用户名 / 密码 **nacos / nacos**



```
sudo mkdir -p /storage/nacos/{data,logs}
sudo chown -R 1000:1000 /storage/nacos
```

nacos-local.yaml

```
# 0. 命名空间（沿用 mid）
apiVersion: v1
kind: Namespace
metadata:
  name: mid
---
# 1. 本地 PV
apiVersion: v1
kind: PersistentVolume
metadata:
  name: nacos-local-pv
spec:
  capacity: { storage: 10Gi }
  accessModes: [ReadWriteOnce]
  persistentVolumeReclaimPolicy: Retain
  storageClassName: local-nacos
  local: { path: /storage/nacos }
  nodeAffinity:
    required:
      nodeSelectorTerms:
      - matchExpressions:
        - key: kubernetes.io/hostname
          operator: In
          values: ["izrj943cxeelu84r2mp9abz"]
---
# 2. PVC
apiVersion: v1
kind: PersistentVolumeClaim
metadata:
  name: nacos-data
  namespace: mid
spec:
  accessModes: [ReadWriteOnce]
  storageClassName: local-nacos
  resources: { requests: { storage: 10Gi } }
---
# 3. ConfigMap（单节点 standalone 模式）
apiVersion: v1
kind: ConfigMap
metadata:
  name: nacos-config
  namespace: mid
data:
  application.properties: |
    # 嵌入式存储，免 MySQL
    spring.datasource.platform=embedded
    nacos.standalone=true
    # 开启 Prometheus 指标
    management.endpoints.web.exposure.include=prometheus
    # 关闭不必要的控制台登录插件（可选）
    nacos.core.auth.enabled=false
    # 日志路径
    server.tomcat.basedir=/home/nacos/logs
---
# 4. Service
apiVersion: v1
kind: Service
metadata:
  name: nacos
  namespace: mid
spec:
  type: NodePort
  ports:
  - port: 8848
    targetPort: 8848
    nodePort: 30848
    name: http
  - port: 9848
    targetPort: 9848
    nodePort: 30849
    name: grpc
  selector: { app: nacos }
---
# 5. Deployment
apiVersion: apps/v1
kind: Deployment
metadata:
  name: nacos
  namespace: mid
spec:
  replicas: 1
  selector:
    matchLabels: { app: nacos }
  template:
    metadata:
      labels: { app: nacos }
    spec:
      nodeSelector: { kubernetes.io/hostname: izrj943cxeelu84r2mp9abz }
      initContainers:
      - name: fix-perm
        image: busybox:1.36
        command: ['sh','-c','chown -R 1000:1000 /home/nacos/data /home/nacos/logs']
        volumeMounts:
        - name: data
          mountPath: /home/nacos/data
        - name: logs
          mountPath: /home/nacos/logs
      containers:
      - name: nacos
        image: nacos/nacos-server:v2.3.2
        ports:
        - containerPort: 8848
          name: http
        - containerPort: 9848
          name: grpc
        env:
        - name: MODE
          value: "standalone"
        - name: NACOS_SERVER_PORT
          value: "8848"
        - name: SPRING_DATASOURCE_PLATFORM
          value: "embedded"
        volumeMounts:
        - name: data
          mountPath: /home/nacos/data
        - name: logs
          mountPath: /home/nacos/logs
        - name: config
          mountPath: /home/nacos/conf/application.properties
          subPath: application.properties
        livenessProbe:
          httpGet:
            path: /nacos/v1/console/health/liveness
            port: 8848
          initialDelaySeconds: 30
          periodSeconds: 10
        readinessProbe:
          httpGet:
            path: /nacos/v1/console/health/readiness
            port: 8848
          initialDelaySeconds: 5
          periodSeconds: 5
        resources:
          requests: { cpu: "200m", memory: "512Mi" }
          limits: { cpu: "1", memory: "1Gi" }
      volumes:
      - name: data
        persistentVolumeClaim: { claimName: nacos-data }
      - name: logs
        emptyDir: {}
      - name: config
        configMap: { name: nacos-config }
```
